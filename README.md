## Installation[⬆](#table-of-contents)
### Node.js[⬆](#table-of-contents)
1. Install typescript globally
  `npm install -g typescript`
2. Install modules
  `npm install`
3. Run typescript
  `tsc` or `tsc --watch`
4. Add Enviroment Variables
  - PORT
  - sendgridapikey
  - sedinblueapikey
5. Run the app
  `npm start`
